# Support

Use this repository for downloads, installation instructions, agent guidance, and release metadata.

For help:

- Clients and implementers should use their NxGN implementation, support, or account contact.
- Internal NxGN teams should use the normal Capstone engineering/support channel.
- Security concerns should follow [SECURITY.md](SECURITY.md).

When requesting support, include:

- tool name: CLI or MCP;
- version from `cap version --json` or the release tag;
- operating system and shell;
- Capstone API URL environment or config value, with tenant/customer details redacted as needed;
- the command or Claude Desktop MCP configuration being used;
- relevant error output.

Do not post credentials, access tokens, private tenant exports, or customer data in public issues or comments.
